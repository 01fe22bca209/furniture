Add category images here to show on the Home page Categories section.

  Sofa.jpg / Sofa.png   - Sofas
  Bed.jpg / Bed.png     - Beds
  Chair.jpg / Chair.png - Chairs
  Table.jpg / Table.png - Tables
  Cabinet.jpg / .png    - Cabinets
  Desk.jpg / Desk.png   - Desks
  Other.jpg / Other.png - Others

If a file is missing, the site will use a default placeholder image.
Recommended size: at least 200x200 px for best display.
